Text Styling Attributes — ID & CLASS Selectors

This page demonstrates common text styling properties. IDs (prefixed with #) are used for single elements; classes (prefixed with .) are reusable. All examples use simple, clear names.

Font family, size, weight, style

#id-font — Georgia, 20px, bold italic (ID example)

.cls-font-small — small text (class)

.cls-font-large .cls-light — large but light weight (class)

Decoration & Transform

#id-decoration — underlined (custom style)

.cls-upper — UPPERCASE

.cls-cap — Capitalize Each Word

Spacing & Line-height

#id-spacing — letter-spacing, word-spacing, and higher line-height help readability when text has more space.

.cls-tight — slightly tighter letters

Alignment & Indent

#id-align — justified text with first-line indent. This simulates a paragraph in a book or article. Long lines will wrap accordingly to show the indent.

.cls-center — centered text

.cls-right — right aligned text

Color, Background & Shadow

#id-color — Blue color with lightblue background

#id-shadow — example with text shadow

.cls-muted — muted (grey) text
